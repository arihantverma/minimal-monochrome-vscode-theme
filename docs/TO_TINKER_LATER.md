# Colors Not Yet Implemented (To Tinker Later)

These colors are intentionally not included to let VS Code defaults take precedence. Add them to the theme files if you want to customize these UI elements.

## Welcome Page

| Property | UI Element |
|----------|------------|
| `welcomePage.background` | Main background of the welcome/get started page |
| `welcomePage.tileBackground` | Background of clickable tiles (Recent, Start, etc.) |
| `welcomePage.tileBorder` | Border around each tile |
| `welcomePage.progress.foreground` | Progress bar color in walkthroughs |
| `walkThrough.embeddedEditorBackground` | Background of code snippets in walkthroughs |

## Settings Editor

| Property | UI Element |
|----------|------------|
| `settings.headerForeground` | Section header text color |
| `settings.modifiedItemIndicator` | Blue bar indicating modified settings |
| `settings.dropdownBackground` | Background of dropdown settings |
| `settings.dropdownForeground` | Text color in dropdown settings |
| `settings.dropdownBorder` | Border of dropdown settings |
| `settings.checkboxBackground` | Background of checkbox settings |
| `settings.checkboxForeground` | Checkmark color |
| `settings.checkboxBorder` | Border of checkboxes |
| `settings.textInputBackground` | Background of text input settings |
| `settings.textInputForeground` | Text color in settings inputs |
| `settings.textInputBorder` | Border of text input settings |
| `settings.numberInputBackground` | Background of number inputs |
| `settings.numberInputForeground` | Text color in number inputs |
| `settings.numberInputBorder` | Border of number inputs |

## Extension Buttons

| Property | UI Element |
|----------|------------|
| `extensionButton.prominentForeground` | "Install" button text |
| `extensionButton.prominentBackground` | "Install" button background |
| `extensionButton.prominentHoverBackground` | "Install" button hover |

## Charts (used in Extensions, Git Graph, etc.)

| Property | UI Element |
|----------|------------|
| `charts.foreground` | Default chart text/line color |
| `charts.lines` | Chart axis lines |
| `charts.red` | Red in pie/bar charts |
| `charts.blue` | Blue in charts |
| `charts.yellow` | Yellow in charts |
| `charts.orange` | Orange in charts |
| `charts.green` | Green in charts |
| `charts.purple` | Purple in charts |

## Previously Listed (Now Implemented)

The following have been added to the theme with pastel semantic colors:

- Bracket Pair Colorization (`editorBracketHighlight.foreground1-6`)
- Notifications (`notifications.*`, `notificationCenter.*`)
- Quick Input / Command Palette (`quickInput.*`)
- Testing Icons (`testing.iconPassed/Failed/Skipped/Queued/Unset`)
- Gutter Indicators (`editorGutter.added/modified/deleted`)
- Merge Conflict Colors (`merge.*`)
- Inlay Hints (`editorInlayHint.*`)
- Sticky Scroll (`editorStickyScroll.*`)
- Diagnostic Backgrounds (`editorError/Warning/Info.background`)
